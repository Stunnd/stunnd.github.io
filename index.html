Stunnd

A flexible CSS grid for designers.

What exactly is Stunnd?

Stunnd is a designer-friendly CSS grid that enables you to rapidly prototype websites. Stunnd is a flexible grid because you can use it as a fixed-width grid if you want, or as a full bleed stretched out, fluid grid if you want. This way, you don't limit yourself to either philosohpy and you can enjoy the pixel perfection of a fixed width grid while using a fluid grid under the hood.

What browsers does Stunnd support?

Technically, Stunnd supports from IE 6, but some features like rounded corners and drop shadows are available only from IE8 onwards. Stunnd supports mobile browsers, too.
