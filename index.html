What exactly is Stunnd?

Stunnd Premium is a designer-friendly CSS grid that enables you to rapidly prototype websites. Stunnd is a flexible grid because you can use it as a fixed-width grid if you want, or as a full bleed stretched out, fluid grid if you want. This way, you don't limit yourself to either philosohpy and you can enjoy the pixel perfection of a fixed width grid while using a fluid grid under the hood.

As of October 2013, Stunnd is production-ready and is already being used on a couple of production websites.

What browsers does Stunnd support?

Technically, Stunnd supports from IE 6, but some features like rounded corners and drop shadows are available only from IE 8 onwards. Stunnd supports mobile browsers, too.

Dependencies

Stunnd Premium is just a templating Grid based on CSS. Due to licensing restrictions, we are not able to include the CSS3 PIE HTC file required for Internet Explorer version 8 and below to render rounded corners and drop shadows correctly. There you will be required to download them separately and use at your own risk. Don't forget to add the HTC file to a subfolder titled 'scripts' in your project folder, since the project files do contain a reference to '/scripts/PIE.htc.' Obviously you will also need additionally a CSS reset file, a typography CSS and a print CSS depending on the natur of your project. However, it is to be noted that the Stunnd files will also work without these.

There will be support for Mobile, Reset, Typography and Print in the coming versions soon.
